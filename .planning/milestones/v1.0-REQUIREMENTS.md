# Milestone v1.0 Requirements Archive

**Archived:** 2026-03-12
**Milestone:** v1.0
**Outcome:** shipped

## Validated v1 Requirements

### Calculator Core

- [x] CALC-01: Form chiaro per temperatura, tempo, quantita di farina e unita.
- [x] CALC-02: Aggiornamento live senza submit.
- [x] CALC-03: Grammi di lievito fresco per la ricetta + equivalente pratico di lievito secco istantaneo.
- [x] CALC-04: Preset rapidi che toccano solo tempo e temperatura.
- [x] CALC-05: Reset completo allo stato iniziale.

### Reliability and Guidance

- [x] SAFE-01: Gestione robusta di vuoti, errori e input non calcolabili con sospensione pulita del risultato.
- [x] SAFE-02: Warning non bloccante fuori intervallo empirico.
- [x] SAFE-03: Rounding pratico senza falsa precisione.

### Model Integrity

- [x] MODEL-01: Modello attivo calibrato contro fixture esplicite.
- [x] MODEL-02: Riallineamento senza rompere contratto, warning, monotonicita e continuita.

### Model Transparency

- [x] TRAN-01: Formula base visibile e variabili spiegate.
- [x] TRAN-02: Disclaimer chiaro sul carattere pratico della stima.
- [x] TRAN-03: Riepilogo leggibile di parametri attivi e correttivi sperimentali.

### Experience and Presentation

- [x] UX-01: UI premium leggibile e coerente su mobile e desktop.
- [x] UX-02: Separazione chiara tra input, risultato, modello e sezione avanzata.
- [x] UX-03: Interazione rapida e stabile con motion sobrie.

### Experimental Extension Seam

- [x] EXP-01: Sezione separata per correttivi sperimentali, chiusa di default.
- [x] EXP-02: Fattori futuri visibili senza influire sul calcolo attivo.

### Engineering Quality

- [x] ENG-01: Comandi npm chiari per lint, typecheck, unit, integration, e2e e build.
- [x] ENG-02: CI GitHub-ready ripetibile.

## Requirement Outcomes

- Nessun requisito v1 e stato scartato.
- CALC-03 e stato affinato in corso d'opera: da riferimento per 1 kg a equivalente pratico di lievito secco istantaneo per la ricetta reale.
- MODEL-01 e MODEL-02 sono nati come inserimento urgente 03.1 dopo la scoperta del disallineamento numerico del modello iniziale.
